# PDF-Reader-Chatbot
The PDF Reader Chatbot is able to read the PDF provided by the user and then answers the user's questions based on that PDF.
It is great for asking questions from user manuals and guidelines in PDF format.

## Technologies
- JavaScript, HTML, CSS
- PDF.js library
- Google Gemini API

## How It Works
1. Upload a PDF
2. Ask questions about the content
3. Get AI-generated answers
